# neural-MI-estimate-clean


Preliminary code for 'Neural Mutual Information Estimation with Vector Copulas'

Implemented estimators:

- MINE
- InfoNCE
- MRE
- DoE
- MIENF
- MINDE
- SMILE
- VCE (ours)


Included benchmarks:

- Wrapped Gaussian
- Mixture of Gaussians
- Multivariate Student-t
- Swiss Roll 
- Spiral transformation
- Image-like dataset
- Biological sequence (coming soon)
